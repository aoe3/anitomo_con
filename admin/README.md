# Admin Tooling (Private)

This directory contains **private, offline tooling** used to prepare data
for the public scavenger hunt app.

**Nothing in this directory should be deployed to attendees.**

## Responsibilities

- Generate opaque vendor tokens
- Maintain vendor-to-token mappings
- Generate QR code images
- Export public JSON files consumed by the app

## Non-Goals

- No live backend
- No vendor accounts
- No attendee tracking
- No personal data collection
